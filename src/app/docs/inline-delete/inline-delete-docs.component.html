The inline delete confirmation appears when users delete items in lists and requires them to confirm that they want to delete the items.

Use the inline delete confirmation when users need to confirm that they want to delete an item from a list.

Don't use inline delete confirmation when deleting an item from outside of a list. Use a confirm dialog instead.

Delete button Cancel button Overlay

When the user invokes the delete action the inline delete confirmation fades in to cover the list item. When the user confirms the delete remove the item and slide the remaining items into their new positions.
